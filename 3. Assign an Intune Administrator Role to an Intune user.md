# Assigning the Intune Administrator Directory Role to an Existing User

## Introduction

This session explains how to assign the **Intune Administrator directory role** to an existing Intune user account. Directory roles define the set of permissions granted to users within Microsoft Intune and Azure Active Directory.

Roles determine what actions a user can perform, such as managing devices, configuring policies, or administering Intune settings.

---

## Understanding Directory Roles in Intune

* A **directory role** defines permissions assigned to a user.
* Microsoft provides:

  * **Built-in roles** for common Intune scenarios
  * **Custom roles** that can be created with specific, granular permissions
* Built-in roles are sufficient for most administrative use cases.
* Roles can be managed using:

  * Azure Active Directory Admin Center
  * Microsoft Endpoint Manager Admin Center

In this demonstration, the **Microsoft Endpoint Manager Admin Center** is used.

---

## Scenario Covered

The objective is to assign the **Intune Administrator directory role** to an existing Intune user named **Test User1**.

---

## Accessing Microsoft Endpoint Manager Admin Center

1. Open a web browser.
2. Navigate to:
   [https://endpoint.microsoft.com](https://endpoint.microsoft.com)
3. Sign in using a **Global Administrator** account.

> A Global Administrator account is required to assign directory roles.

---

## Navigating to Users

1. On the home page, click **Users**.
2. You are taken to the **All Users** page.
3. This page displays a list of all existing users in the tenant.

---

## Selecting the User

1. Locate and click on the user **Test User1**.
2. The user profile page opens.

---

## Viewing Assigned Roles

1. Under the **Manage** section, click **Assigned roles**.
2. A message appears indicating:

   * No directory roles are currently assigned to this user.

---

## Assigning the Intune Administrator Role

1. Click **Add assignments**.
2. In the role search field, search for **Intune Administrator**.
3. Select **Intune Administrator** from the list.

---

## Role Description and Permissions

The **Intune Administrator** directory role:

* Can manage **all aspects of the Intune product**
* Has permissions equivalent to an Intune Global Administrator
* Does **not** have permission to create or manage other administrators with directory role options

---

## Confirming the Assignment

1. Click **Add**.
2. Wait for the confirmation message.
3. A success message appears confirming:

   * The Intune Administrator role has been successfully assigned.

---

## Verifying the Role Assignment

1. Click **Refresh**.
2. The **Intune Administrator** directory role now appears under assigned roles for **Test User1**.

---

## Final Result

* The user **Test User1** now has the **Intune Administrator directory role**.
* The user can manage Intune settings, devices, and policies according to the roleâ€™s permissions.

---

## Conclusion

This completes the process of assigning the **Intune Administrator directory role** to an existing Intune user account using the Microsoft Endpoint Manager Admin Center.
