# Lecture Notes: Manually Enrolling Windows 11 into Intune Using Company Portal

## Overview

* **Topic:** Manual enrollment of Windows 11 devices into Microsoft Intune using the **Company Portal app**
* **Use Case:** Enroll both personal and corporate-owned Windows 11 devices for Intune management
* **Result:** Device is managed by Intune, providing secure access to organizational apps and resources

---

## Key Concepts

* **Company Portal Enrollment**

  * Users can self-enroll Windows 11 devices using the **Company Portal app**
  * Supports both **personal (BYOD)** and **corporate-owned devices**
  * Provides secure access to organizational resources: email, files, and apps

* **Alternative Enrollment Methods**

  * **Settings app** → “Enroll only in device management”
  * **Automatic enrollment** via Group Policy (requires Azure AD join)

---

## Prerequisites

1. **Microsoft Intune subscription** – required to manage devices
2. **User account with Intune license** – must sign in during enrollment
3. **Internet connection** – required throughout the enrollment process

---

## Step-by-Step: Enrollment Using Company Portal

### Install Company Portal App

1. Open **Microsoft Store** from taskbar
2. Search for **Company Portal**
3. Click **Get** to download and install
4. Launch the **Company Portal app**

---

### Enrollment Process

1. Sign in with **work or school account**

   * Example: `user2@msfsurat.onmicrosoft.com`
2. Enter **password** and click **Sign In**
3. Confirm **Allow my organization to manage my device** (checked by default)
4. Wait while device registers and policies are applied

---

### Identity Verification and Security

1. Verify identity using **OTP code** sent to registered phone
2. Enter OTP and click **Verify**
3. Set a **PIN** for additional device security

---

### Completion

* Enrollment completes successfully
* Device name visible on **Home page** of Company Portal (e.g., `Win11-CLI03`)
* Device now has secure access to organization applications and services

---

## Verifying Enrollment

### On Device

* **Settings → System → About**
* Confirm device name matches Company Portal
* Device is now managed by Intune

### In Microsoft Endpoint Manager Admin Center

1. Access via URL: `https://endpoint.microsoft.com`
2. Navigate to **Devices → All devices**
3. Observations:

   * Device name: `Win11-CLI03`
   * Managed by: Intune
   * Ownership: Personal

---

## Key Takeaways

* Company Portal app enables **self-enrollment** for personal and corporate devices
* Enrollment provides **secure access to organizational apps and resources**
* Ownership of device is **personal**, suitable for BYOD scenarios
* Device is fully visible and manageable in **Intune portal**, not dependent on Azure AD join

---

## Summary

* Windows 11 devices can be manually enrolled using **Company Portal app**
* Process involves installing the app, signing in, verifying identity, and setting PIN
* Devices appear in **Intune Admin Center** with full management and compliance details
* Provides secure access to organizational services without requiring automatic enrollment
