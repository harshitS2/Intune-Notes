# Lecture Notes: Manually Enrolling Windows 11 in Microsoft Intune Using Company Portal

## Overview

* **Topic:** Manual enrollment of a Windows 11 device into Microsoft Intune using the **Company Portal app**
* **Audience:** Users enrolling **personal (BYOD)** or **corporate-owned** Windows 11 devices
* **Outcome:** Device is enrolled in Intune and managed under **personal ownership**

---

## Introduction & Context

* **Microsoft Intune** allows organizations to manage:

  * Personal (BYOD) devices
  * Corporate-owned devices
* Once enrolled, users can securely access:

  * Organizational email
  * Files
  * Applications
  * Other protected resources

---

## Enrollment Methods for Windows 11

Users can self-enroll Windows 11 devices using **two methods**:

1. **Company Portal app** (focus of this video)
2. **Windows Settings app**

   * Uses **Enroll only in device management** option

> Intune admins can also configure **automatic enrollment** with no user interaction (covered in a separate video).

---

## Prerequisites for Enrollment

Before starting enrollment, ensure:

1. **Microsoft Intune subscription** is available
2. **User account**:

   * Exists in Azure AD
   * Has an **Intune license assigned**
3. **Internet connectivity**:

   * Device must remain connected throughout the process

---

## Step-by-Step: Install Company Portal App

### Launch Microsoft Store

* Open **Microsoft Store** from the taskbar
* Search for **Company Portal**

### Install Company Portal

* Select **Company Portal app**
* Click **Get**
* Download and install the application
* When prompted to sign in to the Store:

  * Click **No thanks** (not required)

## Step-by-Step: Enroll Windows 11 Using Company Portal

### Launch Company Portal

* Click **Launch** after installation completes

### Sign In

* On the Intune enrollment screen:

  * Sign in with **work or school account**
  * Example: `user2@msfsurat.onmicrosoft.com`
* Enter password and click **Sign in**

### Device Management Consent

* Ensure **Allow my organization to manage my device** is checked
* Click **OK**

---

## Enrollment & Security Verification

* Enrollment process begins:

  * Device is registered
  * Policies are applied
* **Multi-Factor Authentication (MFA)**:

  * Enter OTP sent to registered phone number
* **Security Setup**:

  * Set a **PIN** for Windows Hello

> The enrollment process may take a few minutes.

---

## Enrollment Completion

* Message confirms:

  * Windows 11 enrollment in Intune is successful
* Click **Next**
* Home page shows enrolled device information

---

## Device Verification on Windows 11

### Confirm Device Name

* Go to:

  * **Settings → System**
* Verify device name (example):

  * `WIN11-CLI03`

---

## Verify Enrollment in Intune Portal

### Access Microsoft Endpoint Manager

* Navigate to:
  **[https://endpoint.microsoft.com](https://endpoint.microsoft.com)**
* This opens **Microsoft Endpoint Manager admin center**

### Check Device Status

* Home → **Devices**
* Click **All devices**
* Confirm:

  * Device name: `WIN11-CLI03`
  * **Managed by:** Intune
  * **Ownership:** Personal

## Key Takeaways

* Company Portal provides a **simple self-service method** for Windows 11 Intune enrollment
* Suitable for:

  * Personal (BYOD) devices
  * Corporate devices
* Enrollment results:

  * Device managed by Intune
  * Ownership marked as **Personal**
* Secure access to organizational apps and services is enabled

---

## Summary

* Windows 11 devices can be manually enrolled using the **Company Portal app**
* Requires:

  * Intune license
  * Valid user account
  * Internet connection
* Enrollment includes:

  * Authentication
  * MFA verification
  * PIN setup
* Device appears successfully in Intune under **All devices**
