# Lecture Notes: Deploying a Web App Link to Windows 10 Using Microsoft Intune

## Overview

* **Topic:** Adding and deploying a **Web App (Web Link)** to Windows 10 devices using Microsoft Intune
* **Use Case:** Provide users quick access to internal or external web resources via Company Portal and Start menu
* **Result:** A web shortcut is deployed to Windows 10 devices and visible in Company Portal

---

## What Is a Web App in Intune?

* A **Web App** is a client–server application hosted on the web
* The application itself is:

  * Maintained externally
  * Accessed through a browser
* Intune:

  * Creates a **shortcut** to the web app
  * Controls **who can access** it via group assignments

---

## Behavior on Windows Devices

* On **Windows 10**:

  * A shortcut is added to the **Start menu**
* In **Company Portal**:

  * App appears like any other managed application
* Clicking the app:

  * Opens the URL in the default browser
  * Or in a managed browser (if configured)

---

## Prerequisites

* Role required:

  * Global Administrator **or**
  * Intune Service Administrator
* Access to:

  * Microsoft Endpoint Manager
* Windows 10 device:

  * Enrolled in Intune
  * User is part of the assigned group

---

## Admin Portal Used

* **Portal:** Microsoft Endpoint Manager Admin Center
* **URL:** `https://endpoint.microsoft.com`

---

## Step 1: Add a Web App to Intune

### Navigation

1. Sign in to **Microsoft Endpoint Manager Admin Center**
2. Click **Apps**
3. Under **By platform**, select **Windows**
4. Click **Add**

---

### Select App Type

1. In **Select app type**
2. Under **Other**, choose **Web link**
3. Click **Select**

---

## Step 2: Configure Web App Details

### App Information

* **Name:** MSF Webcast Blogs
* **Description:** Click on this web link to visit MSF Webcast blogs
* **Publisher:** Amazon Webcast (example)

---

### Web URL

* **URL:** `https://www.msftywebcast.com`

---

### Managed Browser Option

* **Use a managed browser:** Optional
* If enabled:

  * Link opens in Intune-managed browser
  * Requires managed browser to be installed

---

### App Category (Optional)

* Select one or more app categories
* Helps users easily locate the app in Company Portal

---

### Featured App Option

* Enable **Show this as a featured app**
* App appears prominently on Company Portal home page

---

### Upload App Icon

1. Click **Select image**
2. Upload a logo/icon
3. Icon appears:

   * In Company Portal
   * In Start menu shortcut

Click **Next**

---

## Step 3: Assign the Web App

### Assignment Type

* **Required** (used in this example)

  * App shortcut installs automatically
* Other options:

  * Available
  * Uninstall

---

### Group Assignment

1. Under **Required**, click **Add groups**
2. Select **Test Users** group
3. Click **Select**
4. Click **Next**

---

## Step 4: Review and Create

1. Review all app settings
2. Click **Create**

---

### Confirmation

* Message displayed:
  **Application MSF Webcast Blogs created successfully**

---

## Step 5: Client-Side Verification (Windows 10)

### Manual Sync (Recommended)

1. Go to **Settings → Accounts**
2. Select **Access work or school**
3. Click **Info**
4. Click **Sync**
5. Restart the device

---

### Start Menu Verification

* After restart:

  * Web app shortcut appears in **Start menu**
* Clicking the shortcut:

  * Opens the configured website in browser

---

## Step 6: Company Portal Verification

1. Open **Company Portal**
2. Go to **Downloads & updates**
3. Under **Installed apps**, confirm:

   * MSF Webcast Blogs
   * **Required by your organization: Yes**

---

## Step 7: Admin-Side Monitoring

### App Install Status

1. In Intune Admin Center:

   * Go to **Apps → Windows apps**
2. Select **MSF Webcast Blogs**
3. Under **Monitor**, check:

   * **Device install status**
   * **User install status**

---

### Device-Level Verification

1. Go to **Devices → All devices**
2. Select the Windows 10 device
3. Under **Monitor**, click **Manage apps**
4. Confirm:

   * Installation status: **Installed**
   * Assignment type: **Required**

---

## Key Takeaways

* Web apps provide **lightweight access** to web resources
* No application binaries are deployed
* Ideal for:

  * Intranet portals
  * Internal tools
  * Documentation sites
* Required assignment ensures:

  * Automatic deployment
  * Consistent user experience
* Easy to monitor from Intune Admin Center

---

## Summary

* A web app link was successfully added to Intune
* Assigned as **Required** to a user group
* Shortcut deployed to Windows 10 Start menu
* App visible and tracked in Company Portal
* Deployment status confirmed at both user and device levels
